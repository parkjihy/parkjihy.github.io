Core Features

The Store Management System has been implemented to provide the following core features:

1. Product Management Page

  • Add, edit, and delete products
  • Manage stock quantity for each product

2. Order Management Page

  • Create orders, view order details, and cancel orders
  • Manage order statuses (e.g., payment completed, shipping, delivered)

3. Customer Management Page

  • Customer sign-up, update, and delete functionality
  • View order history for each customer

4. Sales Management Page

  • Provide daily and monthly sales statistics
  • Analyze and report the most popular products

5. Admin Dashboard Page

  • Notifications for out-of-stock products
  • View total sales and new orders at a glance
  • Intuitive system status overview
